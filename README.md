# DecodeLabs Internship

This repository contains internship task submissions for DecodeLabs.

## Repository Structure

```text
DecodeLabs-Internship/
+-- Task1/
|   +-- Dataset for Data Analytics.xlsx
|   +-- Data_Cleaning_Report.docx
+-- Task2/
    +-- EDA Report.xlsx
    +-- EDA_Report_Nikhil_Choubey.docx
```

## Tasks

### Task 1: Data Cleaning

Includes the original dataset and a data cleaning report documenting the cleaning process, issues found, and final outcomes.

Files:

- `Dataset for Data Analytics.xlsx`
- `Data_Cleaning_Report.docx`

### Task 2: Exploratory Data Analysis

Includes the EDA workbook and a written report summarizing the exploratory analysis.

Files:

- `EDA Report.xlsx`
- `EDA_Report_Nikhil_Choubey.docx`

## Notes

- Reports are provided in Microsoft Word format.
- Data and analysis workbooks are provided in Microsoft Excel format.
